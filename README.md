#Gravatar

Request to the [gravatar.com](http://www.gravatar.com) for a globally recognized avatar

```php
    namespace Gravatar;

    $response = new Response(
        new ProfileRequest(
            new Account('krzysiekpiasecki@gmail.com')
        )
    );

```
